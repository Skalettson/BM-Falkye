# Текстуры для скевоморфного дизайна

Эта папка должна содержать следующие текстуры для полноценной работы скевоморфного интерфейса:

## Деревянные текстуры
- `wood_dark_oak.png` - Тёмный дуб (для панелей и стола)
- `wood_oak.png` - Обычный дуб (для светлых панелей)
- `wood_panel.png` - Деревянная панель с текстурой
- `wood_table_top.png` - Поверхность деревянного стола (64x64 или больше, с тайлингом)

## Металлические текстуры
- `metal_bronze.png` - Бронза (для рамок)
- `metal_gold.png` - Золото (для легендарных элементов)
- `metal_iron.png` - Железо (для декоративных элементов)
- `metal_frame.png` - Металлическая рамка
- `metal_ornament.png` - Металлический орнамент

## Кожаные текстуры
- `leather_dark.png` - Тёмная кожа (для карт и элементов интерфейса)
- `leather_brown.png` - Коричневая кожа
- `leather_panel.png` - Кожаная панель

## Текстуры карт
- `card_back_leather.png` - Рубашка карты (кожаная)
- `card_frame_common.png` - Рамка обычной карты
- `card_frame_rare.png` - Рамка редкой карты
- `card_frame_epic.png` - Рамка эпической карты
- `card_frame_legendary.png` - Рамка легендарной карты (золотая)
- `card_shadow.png` - Тень карты (для объёма)

## Декоративные элементы
- `corner_ornament.png` - Угловой орнамент (для рамок)
- `divider_metal.png` - Металлический разделитель
- `button_leather.png` - Кожаная кнопка
- `button_metal.png` - Металлическая кнопка

## Эффекты
- `glow_soft.png` - Мягкое свечение
- `glow_intense.png` - Интенсивное свечение

## Рекомендации по созданию текстур

1. **Дерево**: Используйте текстуры с видимыми сучками, волокнами и естественными вариациями
2. **Металл**: Добавьте блики и отражения для реалистичности
3. **Кожа**: Текстура должна иметь естественные складки и неровности
4. **Карты**: Добавьте тени и объём для имитации физического объекта
5. **Размеры**: Большинство текстур должны быть 16x16 или 32x32 для тайлинга, кроме стола (64x64+)

## Референсы
- Hearthstone: деревянный стол, металлические рамки, кожаные карты
- Gwent: реалистичные текстуры материалов
- Старинные карточные игры: физические материалы и тени
